> 做的时候就是按照题目意思去计算，没有啥算法。看到讨论区有人发了一个通过正则来做的题解，很有意思，记录一下：
```javascript
/**
 * @param {number} n
 * @return {string}
 */
var countAndSay = function(n) {
    let prev = '1'
    for(let i = 1; i < n; i++){
        prev = prev.replace(/(\d)\1*/g, item =>`${item.length}${item[0]}`)
    }
    return prev
};
```
